ROBOTLEGS 0.9.5

WHAT IS ROBOTLEGS?

ROBOTLEGS - pure AS3 micro-architecture that is:
  • a MECHANISM for WIRING your objects together
  • FREE of FLEX dependencies
  • USABLE in PURE AS3, FLEX, and AIR applications
  • EASY to learn
  • uses NATIVE FLASH EVENTS for object communication

WHAT DOES ROBOTLEGS DO?

ROBOTLEGS is an AS3 micro-architecture (framework) with a light footprint and limited scope. Simply put, Robotlegs is there to help you wire your objects together. It provides the GLUE that your application needs to easily function in a decoupled way WITHOUT THE PAIN associated with many frameworks currently available. Through the use of automated metadata based dependency injection Robotlegs REMOVES BOILER PLATE code in an application. By promoting loose coupling and avoiding the use of Singletons and statics in the framework Robotlegs can help you write code that is highly TESTABLE.
